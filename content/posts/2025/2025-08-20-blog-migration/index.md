## 迁移步骤

### 安装 Hugo

按照官方给的[教程](https://gohugo.io/getting-started/quick-start/)，在安装 Hugo 之前，需要安装 [Git](https://git-scm.com/)，[Go](https://go.dev/) 和 [Sass](https://gohugo.io/functions/css/sass/#dart-sass)，下面的代码用 Scoop 进行安装：

```powershell
scoop install main/go
scoop install main/sass
scoop install main/hugo-extended
```
